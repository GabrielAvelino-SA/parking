# Parking
The Descrição

### Poutes
POST -> New Reservation in Parking
```
/parking
```

GET -> Details reservations for plate in Parking
```
/parking/<str:plate>
```

PUT -> Payment 
```
/parking/<int:id>/pay
```

PUT -> Check Out
```
/parking/<int:id>/out
```
